# Senior Artificial Intelligence Specialist — GR8 Tech

**Source:** https://job-boards.eu.greenhouse.io/gr8tech/jobs/4784094101

## About the Company

GR8 Tech develops B2B iGaming platforms. The organization operates across 1000+ employees in multiple locations and time zones, delivering integrated technology solutions and operational support for gaming operators.

## Role Purpose

Hands-on Senior AI/Generative AI Engineer to develop and maintain production-grade LLM systems, including internal automation, chatbots, AI assistants, and MCP services already deployed in production.

## Key Responsibilities

- Own LLM systems from architecture design through production support
- Design generative AI architectures including RAG, agents, and tool-serving systems
- Build and maintain production LLM-powered applications
- Develop chatbots and AI assistants with reliable behavior
- Implement RAG pipelines with embeddings and retrieval mechanisms
- Create agent workflows using tool/function calling
- Integrate MCP servers and similar context-serving components
- Monitor LLM behavior, system health, and operational costs
- Design evaluation frameworks for LLM outputs
- Handle production incidents and establish long-term solutions

## Required Qualifications

- Strong software engineering foundation with proven production LLM system development experience
- Hands-on expertise with LLMs, RAG architectures, conversational AI systems, tool/function calling patterns, and context-serving systems
- Experience integrating AI into real-time production products
- Understanding of performance, scalability, and cost optimization
- AWS cloud experience
- Strong engineering fundamentals (testing, debugging, code reviews, observability)

## Preferred Qualifications

- Multi-agent systems
- Open-source LLM fine-tuning
- ML platforms/MLOps background
- Latency-sensitive system operation
- Production incident ownership in AI contexts

## Tech Stack

Python and SQL; Amazon Bedrock, OpenAI, Anthropic, open-source models; LangChain, LangGraph, Pydantic, Langfuse; Qdrant/FAISS/OpenSearch/pgvector; AWS (ECS/EKS/Lambda, S3, OpenSearch); Docker, Git, CI/CD.
